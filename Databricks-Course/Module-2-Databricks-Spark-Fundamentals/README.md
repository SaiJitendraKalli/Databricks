# Module 2: Databricks & Spark Fundamentals (2-4 weeks)

## Overview

This module introduces Databricks platform and Apache Spark fundamentals. You'll learn to navigate the workspace, create clusters, write PySpark code, and use Databricks SQL.

## Learning Objectives

- Navigate Databricks workspace effectively
- Create and manage clusters
- Write PySpark DataFrames code
- Use Spark SQL for data manipulation
- Understand Spark architecture and execution

## Module Contents

### Notebooks

1. **[01-Databricks-Workspace-Basics.ipynb](./01-Databricks-Workspace-Basics.ipynb)**
   - Workspace navigation
   - Notebooks and folders
   - Repos integration
   - Databricks File System (DBFS)

2. **[02-Clusters-and-Compute.ipynb](./02-Clusters-and-Compute.ipynb)**
   - Cluster types (all-purpose vs job clusters)
   - Cluster configuration
   - Auto-scaling and termination
   - Libraries and dependencies

3. **[03-PySpark-DataFrames-Basics.ipynb](./03-PySpark-DataFrames-Basics.ipynb)**
   - DataFrame creation
   - Schema and data types
   - Basic transformations
   - Actions vs transformations

4. **[04-PySpark-DataFrames-Advanced.ipynb](./04-PySpark-DataFrames-Advanced.ipynb)**
   - Advanced transformations
   - Window functions
   - User-defined functions (UDFs)
   - Performance optimization

5. **[05-Spark-SQL.ipynb](./05-Spark-SQL.ipynb)**
   - SQL basics in Spark
   - Catalog and metastore
   - Views and tables
   - SQL vs DataFrame API

6. **[06-Databricks-SQL-Warehouse.ipynb](./06-Databricks-SQL-Warehouse.ipynb)**
   - SQL Warehouse overview
   - Query editor
   - Dashboards and visualizations
   - Query history and optimization
