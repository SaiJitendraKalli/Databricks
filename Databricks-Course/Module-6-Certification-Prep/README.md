## Exam Details

### Databricks Certified Data Engineer Associate

**Format**:
- 45 multiple-choice questions
- 90 minutes
- Online proctored
- Passing score: ~70%

**Topics Covered**:
- Databricks Lakehouse Platform (24%)
- ELT with Spark SQL and Python (29%)
- Incremental Data Processing (22%)
- Production Pipelines (13%)
- Data Governance (12%)

**Cost**: ~$200 USD

**Validity**: 2 years
