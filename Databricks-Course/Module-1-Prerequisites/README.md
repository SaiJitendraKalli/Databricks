# Module 1: Prerequisites (1-3 weeks)

## Overview

This module refreshes the foundational skills needed for Databricks data engineering. Focus on Python, SQL, and core data engineering concepts.

## Learning Objectives

- Master Python essentials for data engineering
- Review SQL fundamentals and advanced queries
- Understand data engineering core concepts
- Set up your Databricks environment

## Module Contents

### Notebooks

1. **[01-Python-Fundamentals.ipynb](./01-Python-Fundamentals.ipynb)**
   - Python basics refresher
   - Functions and OOP
   - Virtual environments
   - Libraries for data engineering

2. **[02-SQL-Essentials.ipynb](./02-SQL-Essentials.ipynb)**
   - SQL basics and advanced queries
   - Joins, window functions, CTEs
   - Aggregations and subqueries
   - Performance considerations

3. **[03-Data-Engineering-Concepts.ipynb](./03-Data-Engineering-Concepts.ipynb)**
   - Batch vs Streaming
   - Data Warehouse vs Data Lake vs Lakehouse
   - Partitioning strategies
   - Slowly Changing Dimensions (SCD)
   - Medallion architecture intro

4. **[04-Environment-Setup.ipynb](./04-Environment-Setup.ipynb)**
   - Databricks account setup
   - Workspace navigation
   - Cluster creation
   - First notebook

## Time Estimate

- **Beginner**: 3 weeks (15-20 hours)
- **Intermediate**: 1-2 weeks (8-12 hours)
- **Advanced**: 1 week (4-6 hours)
